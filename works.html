01 / We Bare Bears the Puzzle
월드맵 기획 · 미션 연출 · 시나리오

위베어 베어스 IP 기반 퍼즐 게임 월드맵 스토리 구조 총괄. 게임플레이 동선과 서사를 하나의 흐름으로 연결하는 구조 설계 담당.

데이터 구성

퀘스트 데이터

퀘스트 데이터를 JSON으로 구조화하여 기획과 구현 분리. 퀘스트 ID, NPC 참조, 시작 조건, 목표 단계, 보상까지 모든 정보를 데이터로 관리.

필드타입설명
idstring퀘스트 고유 ID
titleIdstring로케일 시스템 참조 제목 ID
providingNPCIdstring퀘스트를 제공하는 NPC ID
completionNPCIdstring퀘스트 완료를 처리하는 NPC ID
startConditionsarray퀘스트 시작 조건 목록
objectivesarray퀘스트의 목표 단계 리스트
rewardsarray퀘스트 완료 보상 목록

startConditions로 선행 퀘스트 완료, 플래그 값 검사, 아이템 보유 여부 등 복수 조건 설정 가능. objectives는 순차 진행되며, 지역 도달(EnterArea), 대화 완료(CompleteDialogue), 적 처치(DefeatEnemies), 아이템 수집(CollectItems) 네 가지 타입 지원.

{
  "id": "quest_potatostew_01",
  "titleId": "Quest_title_potatostew",
  "providingNPCId": "npc_roy",
  "completionNPCId": "npc_roy",
  "startConditions": [],
  "startDialogueId": "conv_potatostew_01",
  "completionDialogueId": "conv_potatostew_05",
  "objectives": [ ... ],
  "rewards": [
    { "itemId": "Item_0001", "amount": 100 },
    { "itemId": "Item_0002", "amount": 100 }
  ]
}

대화 데이터

캐릭터 간 텍스트 대화를 데이터로 제어. 캐릭터 이름, 초상화(감정), 음성, 애니메이션 연출을 각 대사 라인별로 설정 가능하며, 대화 종료 시 플래그 변경을 통해 스토리 및 퀘스트 진행 제어.

필드타입설명
textIdstring텍스트 대사 로케일 ID
characterIdstring대사를 하는 캐릭터 ID
animationstring캐릭터 애니메이션
expressionstring캐릭터 초상화 ID
voicestring캐릭터 음성 ID
flagActionsarray대화 종료 후 변경할 플래그

flagActions는 네 가지 액션 타입 지원: 플래그 설정(0), 값 증가(1), 플래그 토글(2), 플래그 제거(3). 대화만으로도 퀘스트 상태 전환, NPC 행동 변화 등 유연하게 제어 가능.

{
  "id": "conv_roy_default",
  "lines": [
    {
      "characterId": "npc_roy",
      "expression": "idle",
      "textId": "dlg_roy_default_dlg_01",
      "voice": "talk",
      "animation": "Talk_1",
      "flagActions": [
        {
          "type": 0,
          "key": "potatostew_phase",
          "value": 7
        }
      ]
    }
  ]
}
데이터 구성

캐릭터 데이터

NPC의 표정, 음성, 기본 대화, 플래그 조건부 대화를 데이터로 관리. dialogueConditions를 통해 플래그 값에 따라 NPC 재생 대화를 자동 전환, 별도 코드 없이 NPC 상태 변화 표현.

필드타입설명
characterIdstring캐릭터 고유 ID
nameKeystring이름의 로케일 키
defaultDialogueIdstring조건 없을 때 기본 재생 대화 ID
greetingLocaleIdstringNPC에게 말 걸었을 때 인삿말 로케일
expressionsarray감정 목록 (key + address)
voicesarray음성 목록 (key + address)
dialogueConditionsarray플래그 값에 따라 재생될 대화 조건
 
{
  "characterId": "npc_roy",
  "nameKey": "npc_01_name",
  "defaultDialogueId": "conv_roy_default",
  "greetingLocaleId": "NPC_01_interact_01",
  "expressions": [
    { "key": "idle",     "address": "Roy_idle" },
    { "key": "thinking", "address": "Roy_think" }
  ],
  "voices": [
    { "key": "talk", "address": "Roy_talk" }
  ],
  "dialogueConditions": [
    {
      "flagId": "potatostew_phase",
      "state": 1,
      "dialogueId": "conv_potatostew_10"
    }
  ]
}
